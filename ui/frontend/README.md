# WEION - Enterprise Cognitive System

The **Enterprise Cognitive Operating System**. Autonomous, auditable, and aligned with human values.

## Prerequisites

- Node.js 18.17 or later
- npm or yarn

## Getting Started

1. **Navigate to the project directory:**
   ```bash
   cd ui/frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the application:**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
  - `layout`: Shells, sidebars, headers.
  - `ui`: Primitive components (Buttons, Inputs, Cards).
- `src/lib`: Utility functions.

## Key Features

- **Neo-Enterprise Design**: Matte black/zinc aesthetic with high-contrast typography.
- **Autonomous Agents**: Simulation of reasoning and execution loops.
- **Governance**: Constitutional AI and human-in-the-loop workflows.
